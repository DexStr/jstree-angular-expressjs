Binding events to jsTree

Steps :
  1. Download jsTree.directive.js OR run bower install jstree-directive
  2. Include the reference in your index.html
  3. Add 'jsTree.directive'as a dependency to your app and you can bind events to your tree
You can find a complete list of events here.

Add Single Event to the Tree

Set tree-events="eventName:ScopeCallback" 

<js-tree tree-data="json" tree-src="../jstree/data/tree-json.json" tree-events="ready:readyCB"></js-tree>

The callback functions are defined in the controller scope 
$scope.readyCB = function() {
      console.log('ready event call back');
    };

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])

Demo

PS : Open developer console to view the logs

Add Multiple Events to the Tree

Set tree-events="eventName:ScopeCallback;eventName:ScopeCallback" 

<js-tree tree-data="json" tree-src="../jstree/data/tree-json.json" tree-events="ready:readyCB;changed:changedCB;open_node:openNodeCB"></js-tree>

The callback functions are defined in the controller scope 
$scope.readyCB = function() {
      console.log('ready event call back');
    };
$scope.changedCB = function(e, data) {
      console.log('changed event call back');
    };
$scope.openNodeCB = function(e, data) {
      console.log('open-node event call back');
    };

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])
