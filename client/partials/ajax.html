Loading jsTree via AJAX

This example is powered by Expressjs

Include jsTree.directive.js in your index.htmland add any of the below configurations to load a Tree via AJAX

Tree built with Expressjs - AJAX

Set tree-ajax="/tree" where /tree is the server endpoint 

<js-tree  tree-ajax="/tree"></js-tree>

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])

Demo

AJAX Loading & Plugins [checkbox, dnd]

Set tree-ajax="/tree" and tree-plugins="checkbox,dnd" 

<js-tree  tree-ajax="/tree" tree-plugins="checkbox,dnd"></js-tree>

Angular Directive Integration 
var app = angular.module('treeDemo', ['jsTree.directive'])
